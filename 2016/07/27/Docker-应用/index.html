Docker Compose编排

Docker Compose是用于定义和组装运行多容器分布式应用的工具，
它提供一个简单的基于YAML语言的docker－compose.yml配置文件。 通常，我们使用docker定义和运行复杂的应用，
使用docker compose，在一个文件里定义多容器应用的启动顺序，起到服务编排的作用。
